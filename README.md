# Bonout web-app

## Installation

**Node.js 16.15.0 is required**

> Recommended Node.js version: 18 (LTS)

```sh-session
npm install
```

## Development

```sh-session
npm run dev
```

And then open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

## Deployment

Vercel and Planetscale!
